## Foundry

**This a vault made for the Predexyo Solidity Test**

The Repo consists of:

-   **Vault.sol**: A vault contract that allows users to deposit eth and wrap it and deposit erc20s.
-   **Vault.t.sol**: A test for the vault contract.


## Usage

### Build

```shell
$ forge install
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```
